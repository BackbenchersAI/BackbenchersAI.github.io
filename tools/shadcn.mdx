## ✨ Features

- ⛓️ Built on Radix UI primitives
- 🎨 Fully customizable with Tailwind CSS
- 🧱 Component-driven architecture
- 🌙 Dark mode support out of the box
- 🔌 CLI for scaffolding and customization

---

## 📦 Installation

<CodeBlock language="bash">
npx shadcn-ui@latest init
</CodeBlock>

Choose:

- Framework: `Next.js`
- Language: `TypeScript`
- Styling: `Tailwind CSS`
- App Directory: `Yes`
- Components: Start with `alert`, `button`, `card`, etc.

## ⚙️ Usage Example

### `app/page.tsx`

<CodeBlock language="tsx">
import { Button } from "@/components/ui/button";

export default function Home() {
  return (
    &lt;main className="flex flex-col items-center justify-center h-screen"&gt;
      &lt;Button variant="ghost"&gt;
        Click Me
      &lt;/Button&gt;
    &lt;/main&gt;
  );
}
</CodeBlock>

---

## 🎛️ Customization

Components live in your local `/components/ui` directory and can be fully customized:

<CodeBlock language="tsx">
export const Button = forwardRef&lt;HTMLButtonElement, ButtonProps&gt;(({
  className,
  variant,
  size,
  ...props
}, ref) =&gt; {
  return (
    &lt;button
      ref={ref}
      className={cn("inline-flex items-center rounded-md", className)}
      {...props}
    /&gt;
  )
});
</CodeBlock>

> Tip: Use the `shadcn-ui@latest add <component>` command to scaffold new components.

## 🧰 CLI Utilities

| Command | Description |
|--------|-------------|
| `shadcn-ui init` | Initializes the setup |
| `shadcn-ui add button` | Adds a new UI component |
| `shadcn-ui upgrade` | Updates all local components |

---

## ✅ Recommended Config

- Tailwind v3.4+ with `@tailwindcss/forms`
- Prettier with Tailwind plugin
- Radix UI for accessibility and composability
- Zod for input validation in server actions
